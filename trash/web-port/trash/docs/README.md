# Trash

이 폴더는 삭제 전 보관함이다.

여기로 옮긴 문서는 현재 새 웹포트 재구축의 기준 문서가 아니다. 과거 판단, 실험 과정, 생성 산출물, 또는 깨진 구문 문서를 보존하기 위해 남겨둔다.

## 옮긴 기준

- OpenRouter 기반 자동 포팅 루프 문서
- 기존 훈련 커맨드 이식 태스크와 아키텍처 문서
- `TRAINING_*` 포팅 실험 문서
- `CONTENT_*` 콘텐츠 팩 실험 문서
- 오래된 한글 깨짐 문서
- 대형 생성 산출물 `game_structure.json`
- 현재 새 설계 기준으로 보기 애매한 정책/체크리스트 문서
- `docs/ai-port/` 실험 폴더

## 사용 규칙

- 여기 있는 문서를 보고 새 구조를 결정하지 않는다.
- 필요한 내용이 있으면 현재 기준 문서로 다시 검토해서 옮긴다.
- 바로 삭제하지 않는 이유는 과거 판단 근거와 실험 산출물이 아직 추적 가치가 있을 수 있기 때문이다.
