# Trash

이 폴더는 `web-port` 루트의 휴지통이다.

현재 구현 기준에서 치운 파일과 폴더를 바로 삭제하지 않고 여기에 둔다. 새 설계나 작업 기준으로 사용하지 말고, 과거 근거 확인이 필요할 때만 열어본다.

## 현재 보관물

| 경로 | 내용 |
| --- | --- |
| `docs/` | 과거 문서, AI 포팅 실험 문서, 훈련 이식 문서, 콘텐츠 실험 문서, 깨진 옛 문서, 대형 생성 JSON |
| `artifacts/` | AI 포팅 실행 결과와 재시도 산출물 |
| `analysis/` | 이전 구조 분석 JSON |
| `tools/ai-port/` | OpenRouter 기반 AI 포팅 도구 |
| `src/domain/game/` | 새 방향 확정 전 만든 임시 도메인 모델 실험 |
| `src/legacy/training/` | 소스 폴더 안에 남아 있던 백업 파일 |
| `misc/` | 루트에 있던 기타 찌꺼기 파일 |
