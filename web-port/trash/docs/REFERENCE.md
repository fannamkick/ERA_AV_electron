| ID | 이름 | 설명 | 상태 |

### 커맨드 실행 조건 (COMABLE.ERB)

조건은 복잡한 트리 구조로 되어 있습니다. 주요 조건:

```
COMF0-5 (기본 조교)
└─ 항상 사용 가능

COMF6-19 (고급 애무)
├─ 아이템 보유 필요
└─ 능력치 조건

COMF20-29 (체위)
├─ 특정 의상 착용
├─ 절정 횟수
└─ 신뢰도

COMF60-92 (고급)
├─ 특정 이벤트 완료
├─ 특성 보유
└─ 특수 플래그
```

자세한 조건은 원본 `COMABLE.ERB` 참고 (5,346줄).

## 원본 게임 구조

### 통계
- **총 1,511개 함수**
- **1,461개 호출 관계**
- **369개 ERB 파일**
- **155개 캐릭터**

### 주요 모듈
```
ERB/
├── 시스템관계/        # 캐릭터 정보, 상점, 저장
├── 이벤트관계/        # 오프닝, 엔딩, 일일 이벤트
├── 지도관계/          # 훈련 시스템 (99개 파일)
├── 능력상승관계/      # 능력치 레벨업
├── 매춘관계/          # 매춘 시스템
├── 방문관계/          # 캐릭터 방문
├── 미션관계/          # 미션 시스템
├── 실적관계/          # 업적 시스템
└── AV촬영관계/        # AV 촬영
```

### 핵심 파일
- **COMABLE.ERB** (148KB): 커맨드 사용 가능 조건
- **ZNAME.ERB** (65KB): 캐릭터 이름 시스템
- **EVENT_TRAIN_MESSAGE_A/B.ERB** (200KB): 조교 메시지

## 게임 공략

### 초보자 팁

**1. 초기 설정**
- 난이도: NORMAL 권장
- 목표 금액: 500만 엔 권장
- 주 제한: 52주 권장

**2. 초반 공략**
```
Day 1-7: 기본 조교 (COMF0-5)
  ↓
Day 8-14: 신뢰도 상승 (ABL:10)
  ↓
Day 15-21: 특수 플레이 해금 (COMF20+)
  ↓
Day 22+: 고급 훈련, 매춘, AV 촬영
```

**3. 함락 조건**
```typescript
// 캐릭터 함락 (신뢰도 기준)
if (abl[10] >= 5) {
  // 레벨 5: 고백 이벤트
  // 레벨 8: 완전 함락
  // 레벨 10: 엔딩 해금
}
```

**4. 엔딩 종류**
- 주간 엔딩: 주 제한 내 목표 금액 달성
- 캐릭터 엔딩: 특정 캐릭터 함락 + 조건
- 하렘 엔딩: 복수 캐릭터 함락
- 바드 엔딩: 실패 조건

### 특수 소질 획득

| 소질 | 획득 조건 |
|------|----------|
| C개발 | C절정 10회 |
| V개발 | V절정 10회 |
| 쾌락중독 | 절정 100회 |
| 정액중독 | 사정 받기 50회 |
| 임신 | 질내사정 + 확률 |

### 매력치 시스템
```typescript
// 매력 = BASE:2
// 조교 효율 = 매력 × 0.1
if (base[2] >= 50) {
  efficiency *= 1.5;  // 매력 50 이상 시 효율 +50%
}
```
