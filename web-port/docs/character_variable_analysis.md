# character.ts 변수 변동 추적 문서

> **목적**: 캐릭터 생성 시 초기값 설정 가이드
> - 조교로 변동되는 변수 → 0 또는 낮은 값으로 초기화
> - 변동 안 되는 변수 → 랜덤/고정값으로 캐릭터별 설정 필요

---

## BASE Variables (기본 능력치)

| 변수 | 명칭 | 조교 중 변동 | 이벤트 변동 | 초기값 설정 방법 |
|------|------|-------------|------------|----------------|
| BASE:0 | 체력 | ❌ (LOSEBASE로 감소만) | ✅ (매일 회복) | MAXBASE:0과 동일하게 설정 (예: 1500) |
| BASE:1 | 기력 | ❌ (LOSEBASE로 감소만) | ✅ (매일 회복) | MAXBASE:1과 동일하게 설정 (예: 1500) |
| BASE:2 | 사정게이지 | ✅ (삽입계 커맨드로 증가) | ✅ (사정 시 감소) | 0으로 초기화 |
| BASE:3 | 분유게이지 | ✅ (가슴 자극 시 증가) | ✅ (분유 시 감소) | 0으로 초기화 |
| BASE:7 | 외견연령 | ❌ | ❌ | 캐릭터별 고정값 (예: 18, 25) |
| BASE:8 | 어림연령 | ❌ | ❌ | 고정값 (0: 정확, 1: ~대, 2: ~대 전반) |
| BASE:9 | 진짜연령 | ❌ | ✅ (생일마다 +1) | 캐릭터별 고정값 |
| BASE:10 | 수명 | ❌ | ✅ (매일 -1) | 고정값 또는 0 (무한) |
| BASE:11 | 탄생월 | ❌ | ❌ | 1~12 랜덤 |
| BASE:12 | 탄생일 | ❌ | ❌ | 1~31 랜덤 |
| BASE:13 | 혈액형 | ❌ | ❌ | 0~4 랜덤 (0:불명, 1:A, 2:B, 3:O, 4:AB) |
| BASE:20 | 키 | ❌ | ❌ | 캐릭터별 랜덤 (140~180) |
| BASE:21 | 체중 | ❌ | ✅ (임신 시 증가) | 키 기반 계산 |
| BASE:22 | 가슴 | ❌ | ✅ (임신 시 증가) | 캐릭터별 랜덤 (65~105) |
| BASE:23 | 허리 | ❌ | ✅ (임신 시 증가) | 키 기반 계산 |
| BASE:24 | 엉덩이 | ❌ | ✅ (임신 시 증가) | 키 기반 계산 |
| BASE:30 | 출연료 | ❌ | ✅ (능력치 기반 자동 계산) | 0 (자동 계산됨) |
| BASE:31 | 매력치 | ❌ | ✅ (능력치 기반 자동 계산) | 0 (자동 계산됨) |
| BASE:32~34 | 사이즈 소수 | ❌ | ❌ | 0~9 랜덤 |
| BASE:50 | 페니스 사이즈 | ❌ | ❌ | TALENT:121/122 있으면 설정, 없으면 0 |
